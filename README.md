# 使用pyqt5搭建的文件传送简易版本(部分功能待优化...)
- 可在windows下与mac下运行
###   实现的功能如下：
- 批量发送、下载文件
- 创建目录
- 修改名字
- 递归删除
- 发送、下载时的进度显示
